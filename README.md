# MMSE: Massively Multilingual Safety Evaluation Benchmark

Code and data for the paper:
"MMSE: A Massively Multilingual Benchmark for Typology-Aware 
Multi-Metric Safety Evaluation of Language Models and Guardrails"

NeurIPS 2026 Datasets and Benchmarks Track (Anonymous Submission)

## Contents (to be released upon acceptance)
- Dataset construction pipeline (translation + quality filtering)
- LLM-as-judge evaluation pipeline
- Analysis and visualization scripts
- Judge prompt templates
- Model response generation scripts
